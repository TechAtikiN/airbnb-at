# **AirbnbAt**
## Description
[AirbnbAt](www.google.com) is an Airbnb clone built using Javascript framework **Next.js** and styled using **Tailwind CSS**. 

## Tech Stack
- [x] React.js
- [x] Next.js
- [x] Tailwind
- [x] Pigeon maps API
- [x] Calendar Picker API

## Features
* Pigeon Maps Integration
* Progress bar
* Fully Responsive
* Server Side Rendering
